# Research and Analysis Template

**Context**: Use this template for deep research, data analysis, and information gathering tasks.

**Task**: Research and analyze the current state of [TOPIC] to provide comprehensive insights.

**Research Scope**:
- Current best practices and standards
- Recent developments and trends
- Comparative analysis of available solutions
- Implementation considerations and trade-offs

**Requirements**:
- Gather information from authoritative sources
- Provide factual, evidence-based analysis
- Include concrete examples and use cases
- Consider implementation feasibility and constraints

**Output Format**:
```json
{
  "summary": "Brief overview of findings",
  "key_findings": ["Finding 1", "Finding 2", "Finding 3"],
  "recommendations": ["Recommendation 1", "Recommendation 2"],
  "sources": ["Source 1", "Source 2"],
  "implementation_notes": "Technical considerations"
}
```

**Validation Criteria**:
- All claims supported by sources
- Analysis is balanced and objective
- Recommendations are actionable
- Technical accuracy verified
